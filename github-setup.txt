# GitHub Repository Setup Commands

## Prerequisites
# Make sure you have Git installed and configured
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

## Step 1: Initialize local repository
cd "h:\Projects\userscript_MSTeams_autostatus"
git init

## Step 2: Add all files
git add .

## Step 3: Create initial commit
git commit -m "Initial commit - MS Teams Auto Status Userscript v1.1.0"

## Step 4: Add remote repository (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/zinthose/msteams-auto-status-userscript.git

## Step 5: Push to GitHub
git branch -M main
git push -u origin main

# Alternative for SSH (if you have SSH keys set up)
# git remote add origin git@github.com:YOUR_USERNAME/msteams-auto-status-userscript.git
